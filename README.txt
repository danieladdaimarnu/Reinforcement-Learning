Readme

The source code were created using IPython Notebook provided by Google Colab.

How to run the notebook.
===========================

The notebooks can be viewed using any Jupyter server instance, however, to RUN the notebooks, kindly upload it to Google Colab which already have the Python environment with the required Python packages installed to successfully execute the code.
